# Job Mapping Methodology

## The Universal Job Map (8 Steps)

### 1. Define
What needs to be done? Planning, goal-setting, information gathering

### 2. Locate
Finding what's needed. Searching, identifying sources

### 3. Prepare
Setting up. Gathering tools, configuring environment

### 4. Confirm
Verifying readiness. Checking, testing, validating

### 5. Execute
Performing the core task. The main action

### 6. Monitor
Tracking progress. Observing, measuring, assessing

### 7. Modify
Adjusting approach. Troubleshooting, optimizing, pivoting

### 8. Conclude
Finishing up. Cleanup, storing, sharing, completing

## How to Use

1. Start from Execute (the core action)
2. Work backward to Define (what leads to this?)
3. Work forward to Conclude (what happens after?)
4. For each step, identify: desired outcomes, pain points, unmet needs
