# How Might We (HMW) Framework

## Purpose

Transform opportunity statements into innovation questions that inspire concept generation.

## Structure

**HMW + [verb] + [opportunity focus] + [desired outcome]?**

## Verbs to Use

- Increase, decrease, eliminate, simplify, transform
- Enable, empower, delight, surprise, reassure
- Connect, personalize, automate, integrate

## Process

1. Start with an opportunity (from ODI scoring)
2. Generate 3-5 HMW questions from different angles
3. Select the most promising HMW(s)
4. Brainstorm concepts for each HMW

## Example

Opportunity: Users struggle to compare insurance plans

HMWs:
- HMW make insurance comparison feel effortless?
- HMW eliminate the fear of missing hidden details?
- HMW transform comparison from a chore into confidence?
- HMW personalize plan recommendations without overwhelming?
