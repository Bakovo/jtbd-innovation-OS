# FOCA Interview Framework

## Framework Components

- **Facts**: What objectively happened?
- **Opinions**: What does the user think/believe?
- **Concerns**: What worries the user?
- **Aspirations**: What does the user wish for?

## Extraction Rules

- Facts become evidence anchors
- Opinions reveal mental models
- Concerns reveal anxieties (inertia/anxiety forces)
- Aspirations reveal desired outcomes

## Use When

Interview transcripts contain mixed objective/subjective content. FOCA helps separate what happened from what the user feels about it.
