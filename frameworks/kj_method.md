# KJ Method (Affinity Diagramming)

## Purpose

Organize large amounts of qualitative data into meaningful groups.

## Process

1. Write each observation/quote on a separate card
2. Mix cards randomly
3. Group cards silently (no talking)
4. Create header cards for each group
5. Form super-groups from related groups

## Application in JTBD OS

Use KJ Method when:
- Processing large interview datasets
- Grouping user quotes by theme
- Identifying job clusters
- Discovering emergent patterns

Thematic groups often become:
- Job statements
- Need categories
- Outcome clusters
- Opportunity areas
