# Context Mapping Framework

## Trigger Types

- Temporal triggers: morning, weekend, end of month
- Event triggers: travel, hosting, starting a new job
- Problem triggers: phone breaks, battery dies, storage full
- Aspiration triggers: wanting to learn, wanting to look good

## Context Dimensions Analysis

Ask for each context:
- Is this a recurring or one-time context?
- Does this context affect job performance?
- What is the user trying to achieve in this context?
- What constraints exist in this context?
