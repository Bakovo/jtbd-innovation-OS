# HMW Framework

## HMW Generation Template

HMW [verb] [opportunity focus] for [user] when [context]?

## HMW Angle Categories

1. Eliminate the negative: HMW remove friction?
2. Amplify the positive: HMW enhance what works?
3. Reframe: HMW look at this differently?
4. Simplify: HMW make this effortless?
5. Delight: HMW surprise and delight?
