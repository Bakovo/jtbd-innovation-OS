# Validation Agent Examples

Concept: Auto-Enhance Camera

Assumptions to validate:
- A1: Users find manual editing burdensome
- A2: Users trust AI auto-correction
- A3: Users would share more photos if editing was effortless

Method: Concept testing + short interview (30 min, n=8)
Success criteria: 6/8 participants rate concept 7+/10 on desirability
