# ODI Framework Details

## Scoring Weights (from scoring_config.yaml)

- User Importance: 1.5x
- Dissatisfaction: 1.3x
- Frequency: 1.2x
- Market Size: 1.0x
- Business Value: 1.0x
- Strategic Fit: 0.8x
- Technical Feasibility: 0.7x

## Category Thresholds

- Core Opportunity: Score >= 7.0
- Adjacent Opportunity: Score >= 5.0
- Exploratory Opportunity: Score < 5.0

## Scoring Guidance

- 1-3: Low (minimal evidence, few users)
- 4-6: Medium (some evidence, moderate user base)
- 7-8: High (strong evidence, many users)
- 9-10: Very High (overwhelming evidence, broad impact)
