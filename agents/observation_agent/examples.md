# Observation Agent Examples

Input: Interview transcript - User says phone camera is "fine" but describes:
"I always take 3-4 shots because the first one is usually blurry. Then I use a separate app to edit the colors because the default looks washed out."

Observation:
- Says: Camera is "fine"
- Does: Takes multiple shots + uses third-party editing app
- Gap: User has accepted poor camera quality as normal but invests significant extra effort
- Hidden Need: Photo quality reliability without extra steps
