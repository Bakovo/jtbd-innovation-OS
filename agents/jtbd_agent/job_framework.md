# JTBD Extraction Framework

## The 5T Checklist
Before finalizing any JTBD, verify:
- **True**: Does this reflect a genuine user need?
- **Tacit**: Is this unspoken but real?
- **Touch**: Does it feel personally relevant?
- **Tension**: Is there frustration or desire?
- **Trigger**: What event activates this?

## Job Types to Extract

1. Functional Job: What practical task?
2. Social Job: How does this affect how others see them?
3. Emotional Job: How does this make them feel about themselves?
4. Supporting Job: What sub-tasks support the main job?

## Bad vs Good JTBD

Bad: I want to buy a smartphone with a great camera
(Bad because it includes product solution)

Good: When I am traveling, I want to capture memorable moments that feel professional, so I can share them with family and feel proud of my photography
