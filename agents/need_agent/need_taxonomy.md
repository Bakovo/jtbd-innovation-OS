# Need Taxonomy Reference

## Pain Categories

Barrier: "I can't because..."
- Time barriers: takes too long
- Skill barriers: too complicated
- Resource barriers: too expensive
- Access barriers: not available

Risk: "I'm afraid that..."
- Performance risk: won't work well
- Financial risk: might waste money
- Social risk: might look bad
- Safety risk: might harm me

Consequence: "When I try, this happens..."
- Direct negative: problem gets worse
- Indirect negative: causes new problems

## Benefit Categories

Required: "I need it to at least..."
Expected: "I assume it should..."
Desired: "I wish it could..."
Unexpected: "I didn't expect but love..."
