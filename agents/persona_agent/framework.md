# Persona Building Framework

## Data Sources for Each Dimension

| Dimension | Data Source |
|-----------|------------|
| Identity | Age, occupation, income hints, family status |
| Life State | Current challenges, recent changes, priorities |
| Behavior | Time use, tool usage, decision process |
| Inner State | Frustrations, hopes, fears, values |
| Social | Family, peers, community, social media |
| Category | Past purchases, brand preferences, engagement |

## Evidence Strength Rules

- Strong: 3+ users share trait or same user confirms across multiple statements
- Moderate: 2 users or 1 with indirect evidence
- Weak: Single user or inferred

## Persona Differentiation

If data supports multiple personas, differentiate by:
- Different jobs they are trying to get done
- Different contexts of use
- Different attitudes toward the category
- Different behavior patterns
